# LaunchBase Scanner engineering staging

This repository contains a temporary compiled LaunchBase Stock Count Scanner engineering staging build. It is not the source repository and it is not a production release.

- Icons and manifest colors are temporary engineering placeholders.
- Use only disposable QA data on this staging origin.
- Browser data created here will not transfer automatically to a later production origin.
- The LaunchBase source repository is private.
- This repository does not grant permission to treat this build as an official production distribution.
